# Account Management

## Step by step

- npm install objection knex
- npm install pg
- npx knex init -x ts
- npm run migrate
- npm run seed
- npm run start

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
